# 🧩 要件定義書：模写評価AIアプリ（Drawing Evaluation App）
---
## 1. 概要
### 1.1 背景
絵を練習する多くのユーザーは、自分の模写やイラストの「上達度」や「正確さ」を客観的に把握する手段を持たない。  
本アプリは、AIを用いてユーザーが描いた模写画像を自動的に評価し、改善点を提示することで、効率的な学習を支援することを目的とする。
### 1.2 目的
- 模写やイラスト練習を行うユーザーに対し、AIによる定量的評価を提供する。  
- ユーザーの学習意欲を向上させる。  
- 評価データを蓄積し、個人の上達履歴を可視化する。

## 3. システム概要図
[ユーザー]↓（画像アップロード）[フロントエンド(Web/スマホ)]↓[バックエンド(APIサーバー)]↓[AIモデル（画像評価）]↓[データベース（ユーザー・結果・履歴）]

---
## 4. 機能要件
| 機能カテゴリ | 機能名 | 機能概要 | 優先度 |
|---------------|--------|------------|----------|
| ユーザー管理 | ユーザー登録/ログイン | Google, Emailでログイン | ★★★ |
| 画像アップロード | 模写画像投稿 | JPEG/PNG形式、最大10MB | ★★★ |
| 評価機能 | AI評価スコア算出 | 類似度・構図・線精度などをAIがスコア化 | ★★★ |
| 可視化機能 | スコア結果表示 | 点数とヒートマップで誤差を可視化 | ★★★ |
| 履歴管理 | 評価履歴閲覧 | 過去スコアの推移グラフ | ★★☆ |
| フィードバック | 改善点提示 | 「線がぶれている」「比率がずれている」等を自動表示 | ★★☆ |
| SNS連携 | シェア機能 | X（旧Twitter）などに投稿可能 | ★☆☆ |
| 管理者機能 | 不正画像検出・ログ閲覧 | 公序良俗に反する投稿をブロック | ★★☆ |

## 7. 評価ロジック概要
- **入力**：  
  ユーザーがアップロードした模写画像と、模写元画像（教師データ）
- **出力**：  
  - 類似度スコア (0〜100)  
  - 構図一致率、線精度、色再現度 などのサブスコア  
  - ヒートマップによる差分可視化
- **使用手法**：  
  - CNNまたはViTによる特徴抽出  
  - Cosine Similarityによる類似度計算  
  - OpenCVによる線形補正・正規化処理
